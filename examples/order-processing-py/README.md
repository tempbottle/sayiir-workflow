# Order Processing (Python)

Complete order processing pipeline with validation, parallel payment + inventory check,
and a join step. Demonstrates retries, fork/join, and durable execution.

Matches the [Order Processing tutorial](https://docs.sayiir.dev/tutorials/order-processing-python/).

## Prerequisites

- Python 3.10+

## Run

```bash
pip install -r requirements.txt
python main.py
```
